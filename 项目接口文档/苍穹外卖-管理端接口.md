| id   | integer | 必须     |        | 分类id   | format: int64 |
| name | string  | 必须     |        | 分类名称 |               |
| sort | integer | 必须     |        | 排序     | format: int32 |
| type | integer | 必须     |        | 分类类型 | format: int32 |
